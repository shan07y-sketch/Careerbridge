# Changelog

All notable changes to the CareerBridge project will be documented in this file.

## [1.0.0] - 2026-06-28
### Added
- **Multi-Tenant Portals**: Individualized dashboards and operations for Student, Employer, University, and Administrative roles.
- **AI Intelligence Platform**: Reusable `AIOrchestrator` incorporating Gemini API wrappers, automated query-result caching, and usage tracking audit logs.
- **Real-Time Communications**: Live Socket.IO messaging gateway supporting presence states and typing triggers.
- **Mobile Capacitor Integration**: Native compile-ready structure utilizing platform-aware native storage services and network sync queues.
- **Docker Orchestration**: Root compose file building and orchestrating PostgreSQL, Redis, Backend REST servers, and Nginx SPA client routers.
